# sim-db
Rudimentary database for config files for wfpsim and gcsim in plaintext. 

Open for contribution by anyone, both for applying new sims as well as improving current ones. However, they will be verified for quality before being added. Please also do specify who worked on it, as I'd rather not have sims misattributed to me... albeit that worry may be unnecessary given the sheer amount of characters I'll be slowly including.

Largely Noelle-focused at the moment due to the creator's bias, but meant for any character and any non-trivial sim, including very low DPS ones or non-KQMS. 

## Usage notes ##
1. Most files .config are all plaintext and can be opened with any text editor - in fact, I encourage the usage of characters-copypaste.config for easier wfpsim writing.

2. **SIMS ARE CURRENTLY LARGELY UNOPTIMIZED.** Some are entirely under construction, in fact (marked as WIP).
